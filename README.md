
# Ritual

Minimalist habit tracking application built with Vanilla JavaScript.

---

## Overview

Ritual is a minimalist habit tracking application built with Vanilla JavaScript.

I created this project to strengthen my understanding of core frontend mechanisms such as:
- state management,
- rendering UI from data,
- array methods,
- DOM manipulation,
- localStorage persistence.

The goal was to build something intentionally simple, but still meaningful and aesthetically polished.

Rather than creating another generic “todo app”, I wanted Ritual to feel calm, focused, and intentional.

The idea behind Ritual comes from the belief that growth is usually built through repetition, consistency, and small daily actions.

---

## Features

- Add and delete habits
- Mark habits as completed
- Daily progress counter
- Automatic daily reset
- Random motivational quote system
- localStorage persistence
- Responsive design
- Calm minimalist UI

---

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API

---

## What I Practiced

This project helped me reinforce:
- rendering UI from application state,
- array methods (`map`, `filter`),
- DOM manipulation,
- event handling,
- conditional rendering,
- responsive design,
- state persistence.

---

## Challenges

One of the most interesting parts was implementing the daily reset mechanism.

I practiced:
- working with JavaScript dates,
- `toISOString()`,
- comparing dates,
- handling `null` values from localStorage,
- conditionally updating application state.
